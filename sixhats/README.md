# SlimApp RESTful API

This is a RESTful api built with the SlimPHP framework and uses MySQL for storage.

### Version
1.0.0

### Usage


### Installation


Edit db/config params

Install SlimPHP and dependencies

```sh
$ composer
```
### API Endpints
```sh
$ GET /api/users
$ GET /api/user/{id}
$ POST /api/users/add
$ PUT /api/customer/update/{id}
$ DELETE /api/user/delete/{id}
```
